# Basic Profile Image
As standard within Poggle your profile picture will be fully rounded, with a shadow. As default your profile image will have a `profileShadow` with a `width` and `height` of `10 rem` units.

# Profile Pictures With Just Letters
If you wish you can render just profile pictures with a custom background colour and some letters (max 2), just add text with the within the child property of the component. You can edit the background of the profile
by changing the `profileBGColor` for the background and `profileTxTColor` for the text colour.

<Canvas>
  <Story name="Profile With Just Letters" args={{profileBGColor: "purple", profileTxTColor: "white", width: 8, height: 8, children:"JD"}}>
    {Template.bind({})}
  </Story>
</Canvas>

You can edit the border radius of the profile easily using the `profileRadius` property:
